2.What is a Bootstrap Container, and how does it work?

Ans :-

    Bootstrap Container :-

  • Bootstrap Containers are the most basic layout element in Bootstrap.

  • Bootstrap Containers are very essential and basic building blocks of bootstrap that wrap a page's content.

  • Containers are defined within the container class (.container).


    • Containers are used for many purposes such as :-

  • It requires use with the default grid system.

  • To establish width for the layout to give web content.

  • To provide responsive fixed behavior of any web project.

  • To sets the content margin dealing with the responsive behavior of your layout.

4.What do you know about the Bootstrap Grid System?

Ans :-

  • The Bootstrap Grid System is a powerful, mobile-first flexbox grid system used for building responsive layouts in web development.

  • It provides a flexible and easy-to-use way to create complex grid-based layouts that adapt to various screen sizes and devices.


    • Key features of the Bootstrap Grid System include:-

  • 12-column Layout

  • Responsive Design

  • Container, Row, and Column Structure

  • Column Nesting

  • Offsetting Columns

  • Responsive Breakpoints

  • Utility Classes

6.What is a Button Group, and what is the class for a basic Button Group?

Ans :-

    button Group :-

  • A Button Group in Bootstrap is a component used to group multiple buttons together horizontally or vertically, creating a cohesive and visually appealing interface.

  • Button groups are commonly used when you want to present a set of related actions or options in close proximity to each other.

  • The class for a basic Button Group in Bootstrap is .btn-group. This class is applied to a container element (such as a <div> or <span>) that wraps around the individual buttons.



7.How can you use Bootstrap to make thumbnails?

  • To create thumbnails in Bootstrap, you can use the .thumbnail class along with the appropriate HTML structure.

  • We use the .thumbnail class to style the container for each thumbnail.

  • Inside each .thumbnail container, we have an <img> element for the thumbnail image and a <div class="caption"> for the content associated with the thumbnail.



8.In Bootstrap 4, what is flexbox?

Ans :-

    Flexbox :-

  • In Bootstrap 4, Flexbox is a powerful layout model that allows for the creation of flexible and responsive layouts with ease.

  • Flexbox stands for "Flexible Box Layout" and is a CSS layout module that provides a more efficient way to distribute space and align items within a container.


    • Flexbox in Bootstrap 4 allows you to:-

  • Easily Align Items

  • Flexible Sizing

  • Ordering

  • Responsive Design
